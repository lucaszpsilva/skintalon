# 🔪 SkinTalon.gg

A modern, fully responsive (multiplatform) web application for managing and trading Counter-Strike weapon skins. This project is built from scratch with a strong focus on high performance, clean component architecture, and pixel-perfect fidelity to modern gaming trading platform designs.

---

## 🚀 Technologies Used

* **React** (Component-based UI library for dynamic user interfaces)
* **Vite** (Next-generation, ultra-fast frontend build tool)
* **TypeScript** (Static typing for robust, scalable, and safe codebase)
* **Tailwind CSS** (Utility-first framework optimized for rapid responsive styling)

## 🛠️ Local Development Setup

To clone, run, and test this project locally on your machine, you will need to have [Git](https://git-scm.com) and [Node.js](https://nodejs.org) installed.

From your terminal, execute the following commands:

```bash
# 1. Clone the repository
git clone [https://github.com/lucaszpsilva/skintalon.git](https://github.com/lucaszpsilva/skintalon.git)

# 2. Navigate into the project folder
cd skintalon

# 3. Install the dependencies
npm install

# 4. Start the local development server
npm run dev
